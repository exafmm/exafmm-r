# rtfmm/3dirpp

## Features
1. kernel independent
1. indexed cell
1. regularization
1. periodic boundary condition
1. precomputation

## Requirements
+ BLAS
+ LAPACK
+ FFTW3 
+ CMAKE

## Compilation
```
mkdir build  
cd build  
cmake .. -DCMAKE_BUILD_TYPE=Release
make  
```
